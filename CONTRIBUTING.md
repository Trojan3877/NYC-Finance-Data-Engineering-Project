# 🤝 Contributing Guide

## How to Contribute
1. Fork the repository
2. Create a feature branch
3. Write clean, documented code
4. Add or update tests
5. Submit a pull request

## Code Standards
- PEP8 compliance
- Clear function naming
- Type hints where possible
- Modular design

## Commit Convention
- feat: new feature
- fix: bug fix
- docs: documentation
- refactor: code cleanup
